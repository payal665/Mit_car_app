# Car Price and Mileage Prediction

This project is a Machine Learning web application built using Streamlit.  
It predicts car price and car mileage based on user inputs.

---

## Features

- Car Price Prediction
- Car Mileage Prediction
- Simple user interface using Streamlit

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

---

## Project Files

- app.py → Main application (Price + Mileage)
- mileage_app.py → Mileage prediction (separate)
- final_model.pkl → Price prediction model
- mileage_model.pkl → Mileage prediction model
- requirements.txt → Required libraries
